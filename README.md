# breast-cancer-prediction
A machine learning model for predicting malignant vs benign tumors using Python
## Tools Used
- Python
- Pandas
- Scikit-learn

## Method
- Data preprocessing
- Train-test split
- Logistic Regression model

## Results
The model achieved approximately 96% accuracy on test data.

## Motivation
This project reflects my interest in applying AI to healthcare, particularly in early detection of diseases such as cancer in resource-limited settings.
